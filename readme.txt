== Description ==
Le but est de permettre d'avoir des pages dont le contenu est réservé à certains. 
Pour que ça fonctionne correctement, il faut que ces "certains" soient des users au sens de WordPress. 
Et on veut également pouvoir réaliser une liste des membres, avec certaines infos qui ne figurent pas dans les données standards de user. 

== Notes ==
=== Le plugin theme-my-login ===
* Il sert à orienter l'utilisateur non admin lors du login pour éviter qu'il ne se retrouve perdu dans le tableau de bord alors qu'il souhaitait aller sur une page sp&cifique
* il sert aussi à générer une page profil différente de la page standard WordPress.
